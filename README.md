<p align="center">米游社、原神、星穹铁道、绝区零</p>

---

#### 安装使用(开发中...)
```bash
# 在karin根目录执行
pnpm add karin-plugin-mystool -w
```

#### 附加功能：[原神](../../../Genshin)、[崩坏：星穹铁道](../../../StarRail)、[绝区零](../../../ZZZero)
> [!WARNING]
>请自行选择安装、不安装则无法使用对应功能
